## Phoenix 個人品牌網站（React）

這是一個使用 React + Vite 建立的個人形象網站，定位為數據／CDP 顧問的專業個人頁面，設計重點：

- **RWD 響應式設計**：從手機到桌機皆有良好閱讀體驗
- **一致色系規範**：以深色系技術感為主，搭配藍綠漸層點綴
- **互動效果**：擅長領域色系切換、卡片 hover 動畫、表單聚焦效果
- **圖片比例**：人像區塊採 4:5，成就區圖片採 16:9，適合簡報視覺

### 開發與啟動

```bash
cd personal-website
npm install
npm run dev
```

瀏覽器開啟 `http://localhost:5173` 查看頁面。
